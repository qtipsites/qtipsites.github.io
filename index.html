About

qTip is a Static Site Generator coded in Batch. I know, right? Who even uses that anymore? Anyways, it's featherweight and easy to use. In fact, to get started, just put your Markdown code in in.md and press generate!

This site was made with it!

Install


  • Download the latest version

  • Unzip the folder onto wherever you want (Desktop, User Folder, etc.)

  • Open the root folder, then open mdin, then in.md in your favorite editor (I use Notepad++)

  • When you're finished editing, run generate.bat

  • Select your style

  • Open index.html in your browser!


Qtip-Flavored Markdown

Because there were certain tags every Web Developer should have, I needed to apply my own tags post-conversion! Here they are:

  • Button: --Button-[function]==[Display Text]--/Button

Yes, yes, I know that looks like a glob of ='s and -'s, and that's because, well, it is. I suggest writing it down somewhere...

  • Title of Page: title:[Title]:title

Why couldn't the button syntax be beautiful, like that?
